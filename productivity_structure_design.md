## 📊 시트 간 연결 방식

### 연동 구조
```
생산성 대시보드
  ├─ VLOOKUP(팀원, 팀현황!A:F) → 팀원별 KPI 자동반영
  ├─ SUMIF(프로젝트, 프로젝트관리!B:C) → 진행률 합산
  ├─ IF(자동화!상태=ready, 신뢰도 +0.5%) → 시스템 신뢰도 가중치
  ├─ AVERAGE(기간분석!일일) → 주간/월간 추이 계산
  └─ IF(KPI!달성도 <경고값) → 자동 알림 생성
```

### 계산 수식 예시
1. **팀 전체 효율 = AVERAGE(팀현황!완료율)**
2. **프로젝트 진행률 = WEIGHTED_SUM(Phase, 가중치)**
3. **시스템 신뢰도 = (Phase2A + Phase2B + Phase2C) / 3**
4. **예상 완료 = TODAY() + 평균소요시간 × 남은작업**
