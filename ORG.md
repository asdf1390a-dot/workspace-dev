## 호출 방식

사용자님은 평소처럼 비서(나)에게 텔레그램으로 말씀하시면 됩니다.
필요한 일에 맞게 비서가 해당 부서장을 호출:

| 사용자 요청 | 비서가 호출하는 부서 |
|---|---|
| "이 PPT 번역해줘" | translator |
| "어제 BM 몇 건이었어?" | data-analyst |
| "엑셀 분석해서 트렌드 알려줘" | data-analyst |
| "/assets에 검색 필터 하나 더 추가해줘" | web-builder |
| "본사 보고용 한 줄 요약 부탁" | translator |

## 파일 위치

- 비서 (나): `~/.openclaw/workspace-dev/SOUL.md` + 메모리 + 스킬
- 부서장들: `~/.claude/agents/<name>.md`
- 공유 스킬: `~/.openclaw/workspace-dev/skills/`
- 공유 메모리: `~/.claude/projects/.../memory/`

## 향후 확장

각 부서장에게 더 좋은 도구·스킬·메모리가 쌓이면 능력이 늘어납니다.
필요할 때 부서 추가:

- **HQ 보고** — 주간/월간 보고서 자동 작성
- **회의 정리** — 녹음/메모 → 요약·할 일
- **인사 총무** — 출근, 휴가, 사내 공지
- **재무·구매** — 견적, 발주, 정산
