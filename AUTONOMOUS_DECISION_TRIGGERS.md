## ⚠️ 확인 필수 항목 (질문 → 승인 필요)

### 1. **3개 이상 동시 작업 위임**
- `Agent(web-builder)` + `Agent(evaluator)` + 3번째 이상
- 리스크: Agent 스포닝 실패 → 위임 선언만 남음
- 규칙: 동시 위임 ≤ 2개 (자동), 3개+ 위임 (확인 필수)

### 2. **Production 배포 & 실환경 변경**
- `git push` → main 브랜치
- DB 마이그레이션 실행
- 서비스 재시작/재배포
- 예: "npm run deploy" 자동 실행 불가

### 3. **대사용자 영향 코드 변경**
- 10개 이상 파일 수정
- API 엔드포인트 삭제/이름 변경
- 데이터 스키마 변경
- 예: "라우팅 대대적 리팩토링" → 확인 필수

### 4. **데이터 삭제 & 마이그레이션**
- `DROP TABLE`, `DELETE` 쿼리 실행
- Backup 데이터 삭제
- 프로덕션 DB 수정
- 예: "데이터 정제" → 반드시 확인 후 진행

### 5. **새로운 라이브러리 설치/제거**
- `npm install <pkg>` — 새 dependency 추가
- `npm uninstall <pkg>` — 기존 dependency 제거
- Major 버전 업그레이드

### 6. **장시간 실행 작업 시작**
- 테스트 suite 전체 실행 (>5분)
- 대규모 빌드/컴파일
