# Claude Video Skill

**Purpose:** Analyze, summarize, and extract insights from video content for team projects. Enable automatic transcription and knowledge extraction from video documentation.

**Scope:**
- Transcribe video content automatically
- Generate comprehensive summaries from video materials
- Extract key information with timestamps
- Support video-based documentation and knowledge sharing
- Create searchable video content archives

**Team Members:** All 15 (CEO + 6 Phase 0 + 4 Phase A/B + 5 Phase C)  
**Primary Users:** web-builder, evaluator, general-purpose agents

**Installation:**

```bash
# Option 1: Claude Video Plugin (Recommended)
/plugin install claude-video

# Option 2: Watch.skill Alternative
/plugin install watch.skill
```

**Key Use Cases:**
1. Training video processing for new team members
2. Meeting recording analysis and documentation
3. Product demo understanding and breakdown
4. Process video walkthroughs and procedures
5. Onboarding video library creation
6. Async team learning from recordings

**Features:**
- ✅ Automatic full transcription
- ✅ Timestamp-based chapter extraction
- ✅ Speaker identification (when available)
- ✅ Key takeaway summarization
- ✅ Action item extraction
- ✅ Visual content description
- ✅ Searchable transcript index

**Workflow:**
1. Upload or link video content
2. Plugin processes and transcribes
3. Generates summary with chapters
4. Creates searchable archive
5. Documents action items

**Productivity Impact:**
- Meeting documentation: 80% faster
- Onboarding: Video library reduces 1-week to 2-3 days
- Knowledge preservation: Permanent video-based knowledge base
- Async learning: Team members learn at own pace

**Integration with Other Skills:**
- Works with **agentmemory** for knowledge preservation
- Complements **understand-anything** for rapid learning
- Supports **andrej-karpathy-skills** documentation standards
