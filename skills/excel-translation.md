# Excel 번역 규칙 (번역가 전용)

## 필수 규칙

1. **시트 처리**
   - 기본(원본) sheet 절대 수정 금지
   - 원본 sheet 복사 → 복사본 이름을 `영문본` 으로 변경 → 영문본 sheet에서만 작업

2. **폰트·서식 유지**
   - 링크, 양식 글꼴, 셀 서식 절대 변경 금지
   - 폰트 크기·종류 원본 그대로 유지
   - 행 높이, 열 너비 조정해서 번역 후 텍스트 잘리지 않게

3. **번역 품질**
   - 에러 발생하지 않게 (수식 참조, 링크 등 건드리지 않음)
   - 헷갈리는 전문 용어 → 셀 배경색 또는 글자색 다르게 표기
   - 해당 셀에 주석(Comment) 달아서 원어 + 설명 기재

4. **텍스트 정렬**
   - 번역 후 텍스트가 셀 밖으로 넘치거나 가려지지 않도록
   - 필요 시 텍스트 줄 바꿈(Wrap Text) 적용

## 작업 순서
1. 원본 sheet 복사 → `영문본` 시트 생성
2. `영문본` 시트에서 한국어 텍스트만 영어로 번역
3. 수식·링크·글꼴 확인 (변경 여부 검토)
4. 헷갈리는 용어 색상 표기 + 주석 추가
5. 행열 너비 조정해서 텍스트 가림 없는지 확인
