# Superpowers Skill

**Purpose:** Enable team members to leverage unique individual capabilities and strengths for maximum impact.

**Scope:**
- Identify personal strengths and specialized expertise
- Amplify individual contributions through focused work
- Collaborate across complementary superpowers
- Develop mastery in specific domains

**Team Members:** All 15 (CEO + 6 Phase 0 + 4 Phase A/B + 5 Phase C)

**Key Use Cases:**
1. Specialized task assignment based on individual strengths
2. Knowledge sharing between experts
3. Cross-functional collaboration
4. Building T-shaped skill profiles

**Framework:**
- Secretary: Coordination and autonomy
- Web-Builder: Full-stack engineering
- Evaluator: Quality assurance and compliance
- Data-Analyst: Analytics and insights
- Automation-Specialist: System automation
- Translator: Cross-cultural communication
- Design Specialist: UI/UX and aesthetics
- DevOps Engineer: Infrastructure and reliability
- Memory System Specialist: Knowledge management
- QA Specialist: Testing and verification
- Project Planner: Planning and resource allocation

**Activation Date:** 2026-06-02
