## 🚀 프로세스 (How) - 6단계

### 단계 1: 회사 PC 준비 (45분)
**목표:** 이전할 데이터 완벽 정리

```
├─ Git 상태 확인 (10분)
│  └─ 모든 변경 커밋/푸시 → origin/main 동기화
│
├─ 환경 변수 백업 (15분)
│  └─ .env.local 파일들 암호화 저장 또는 GitHub Secret
│
├─ API 키 검증 (10분)
│  └─ ANTHROPIC_API_KEY, GITHUB_TOKEN, TELEGRAM, Discord 확인
│
└─ 전체 데이터 백업 (10분)
   └─ tar.gz 압축 + 체크섬 계산
```

**체크리스트:** `MIGRATION_PRE_CHECKLIST.md` 참조

---

### 단계 2: Windows 기본 설정 (45분)
**목표:** 필수 소프트웨어 설치 및 기본 환경 구성

```
├─ 필수 소프트웨어 설치
│  ├─ Git for Windows (https://git-scm.com/download/win)
│  ├─ Node.js v22.22.2 (정확한 버전)
│  ├─ npm 10.9.7 이상
│  └─ OpenClaw 최신 버전
│
└─ 폴더 생성
   ├─ $env:USERPROFILE\.openclaw\
   └─ $env:USERPROFILE\.claude\
```

---

### 단계 3: 데이터 전송 (1시간)
**목표:** 회사 PC 데이터 → Windows 노트북

```
USB/클라우드 전송:
├─ 압축 파일 전송 (openclaw_migration_backup_*.tar.gz)
│
├─ 압축 해제
│  └─ 7-Zip 또는 WinRAR로 자동 해제
│
└─ 검증
   └─ 파일 개수 및 크기 확인
```

---

### 단계 4: 설정 복원 및 재설정 (1시간)
**목표:** OpenClaw/Claude Code 정상 작동 준비

```
├─ Git 설정
│  ├─ user.name / user.email 설정
│  ├─ GitHub 인증 (Token 또는 OAuth)
│  └─ git status 확인 (Clean 상태)
│
├─ 환경 변수 복원
│  ├─ .env.local 파일 복구
│  ├─ 각 API 키 검증
│  └─ npm install (의존성 설치)
│
├─ OpenClaw 재시작
│  ├─ openclaw start 실행
│  └─ 설정 검증
│
└─ 자동화 설정
   └─ Windows Task Scheduler (크론 작업)
```

---

### 단계 5: 정합성 검증 (45분)
**목표:** 모든 기능이 정상 작동하는지 확인

```
검증 항목:
├─ Git 저장소 (origin/main 동기화)
├─ 메모리 파일 (체크섬 일치 확인)
├─ API 호출 (Anthropic, GitHub, Supabase)
├─ Discord 봇 (메시지 송수신)
├─ Telegram 봇 (메시지 송수신)
└─ 성능 (메모리/CPU 모니터링)
```
