# Translator AI Agent 역할 재정의 및 업무 재분장

**작성일:** 2026-05-15 18:00 KST

## 현황 분석

### Translator AI Agent의 현재 상태
- **역량:** 한영 양방향 번역, 용어 일관성, 문화적 뉘앙스, UI/UX 번역
- **상태:** 🟢 유휴 (진행률 0%)
- **다음 할당:** Travel Phase 2 설계서 번역 (2026-05-20 이후)

## 5가지 핵심 문제 분석

### 1️⃣ 용어 불일치로 인한 현장 혼선 (가중치 높음)
- 본사 공문과 현장 문서에서 동일 개념을 다른 용어로 표기
- 번역 품질과 무관하게 현장 혼선 유발
- **원인:** 팀 내 용어 기준 합의 부재

### 2️⃣ UI 라벨 약어 오해 (가중치 높음)
- 한국 본사 기준 약어 → 인도 현장 작업자 의미 전달 실패
- 예: "BM" → 현장 모름
- **필요:** UI 라벨 표준화 (약어 → 풀어쓰기 병기)

### 3️⃣ 글로사리 버전 관리 부재
- 제조업 기술용어 글로사리(260줄) 존재하지만 최신화 미흡
- 신규 용어 추가 규칙 불명확
- **필요:** 글로사리 운영 프로세스 정립

### 4️⃣ 월간 일관성 검증 루틴 부재
- 번역 문서는 DB 스키마와 달리 자동 검증 불가
- 월 1회 일관성 검증 필요하지만 미실행
- **효과:** 장기 누적 오류 방지

### 5️⃣ 현장 신조어 필터링 부재
- 현장에서 생겨난 신조어는 글로사리에 없음
- AI가 오역해도 검증 기제 없음
- **역할:** Translator AI Agent만 가능한 가치 영역

## 새로운 역할 정의

### 장기 역할: 팀 내 문서 표준화 담당자

**책임 범위:**
1. **용어 기준 관리** — 팀 전체가 준수할 표준 용어 수립·운영
2. **글로사리 운영** — 기술용어 글로사리 최신화 및 버전 관리
3. **UI/문서 일관성** — 월 1회 이상 일관성 감시
4. **현장 신조어 통제** — 새로운 용어 발굴 → 글로사리 반영
5. **팀 교육** — 신규 팀원 온보딩 시 용어 기준 설명

**보고 라인:**
- 비서 (주간 리포트만)
- Planner AI Agent (설계 시 검토)
- Web-Builder AI Agent (UI 배포 전 검증)

**예상 효과:**
- 팀 내 용어 혼선 30% → 5% 감소
- 현장 오해 사건 월 2~3건 → 0건
- 번역 검수 시간 30% 단축
- 신입 팀원 온보딩 기간 50% 단축
