# 동적 평가 기준 시스템

**목적:** 고정된 평가 기준이 팀의 성장 속도를 따라가지 못하는 문제 해결

**핵심:** 외부 업계 기준 + 팀 성과 데이터 + 사용자 피드백을 통합하여 매월 기준 동적 조정

## 기준 설정 방식

### 1단계: 외부 벤치마크 수집
- 각 도메인의 업계 리더 기준 조사
- Google, Vercel, Kaggle 등 공개 기준 참고
- 월간 변화 추적

### 2단계: 팀 성과 분석
- 지난 1개월 성과 데이터 수집
- 기준 대비 실제 성과 비교
- 개선/악화 추세 분석

### 3단계: 팀 의견 수렴
- 각 팀원의 피드백 수렴
- 실질적 어려움과 제약 사항 반영
- 합의 기반 기준 결정

### 4단계: 신규 기준 공시
- 이전 기준과 신규 기준 비교표 작성
- 변경 이유 명확히 설명
- 팀 전체 공지

## 현재 적용 기준

### Audit System
