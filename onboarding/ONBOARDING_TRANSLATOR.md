## 도움말
- **용어 불명확할 때:** 선임이나 비서에게 즉시 확인 (추측 금지)
- **평가자의 피드백이 많을 때:** 패턴 분석 → 다음 번역에 반영 (학습)
- **긴급 번역이 필요할 때:** 선임에게 우선순위 알림 → 동일 주제 배치 처리
- **용어 추가 필요할 때:** 용어 사전 제안 → 선임 승인 후 등록

## 파일 제출 형식

### 이메일/텍스트 번역
```markdown
【원문】
부품 발주가 지연되고 있습니다. 즉시 조치를 부탁드립니다.

【번역】
We need to expedite the parts order due to delays. Your immediate action is appreciated.

【검증자】
평가자(선임): [이름]
【검증 시간】
2026-05-19 14:30 KST
```

### Excel 번역
```
원본: data/주간업무양식_KO.xlsx
번역본: data/주간업무양식_EN.xlsx
시트: Sheet1 (생산), Sheet2 (기술), Sheet3 (보전), Sheet4 (생산관리)
범위: A1:Z100 (모든 헤더 + 주석)
```

### GitHub 커밋 메시지
```
chore: 번역 — 주간업무양식 KO→EN | Refs: [task_id] | Stage: TRANSLATION

번역자: [이름]
검증자: [평가자 이름] (검증 대기)
범위: 헤더 + 주석 + KPI 용어
```
