# 표준 온보딩 템플릿 — AI 팀원 즉시 투입 가이드

## 원칙
- **Day 0 (30분):** 역할 요약 + 핵심 책임 확인
- **Day 1 (2시간):** 프로젝트 구조 + 기술스택 + 핵심 코드 3개 파일 읽기
- **Day 1 오후:** 첫 작업 배정 → 즉시 진행
- 3일 이내 "혼자 일할 수 있는 수준" 도달

## 구성 요소 (복사-붙여넣기)
```
역할별 온보딩 문서 (ONBOARDING_[ROLE].md)
├─ 역할 정의 (3줄)
├─ 핵심 책임 4가지
├─ Day 1 체크리스트 (4항목, 2시간)
├─ 핵심 참고 문서 5개 (링크)
├─ 첫 과제 (자료 + 예상 소요시간)
└─ 일주일 로드맵
```

## 체크리스트 (온보딩 담당자용)

- [ ] 역할별 온보딩 문서 제공
- [ ] Day 1 09:00 시작 확인
- [ ] Day 1 14:00 첫 작업 배정
- [ ] Day 2 09:00 진행상황 체크
- [ ] Day 3~5 독립 작업 진행
- [ ] 첫 결과물 → 담당 팀원에게 리뷰 의뢰
