## 📞 문제 발생 시

### 즉시 조치
1. 현재 단계에서 **즉시 중단**
2. 오류 메시지 **스크린샷 촬영**
3. 해당 문서에서 **"문제 해결"** 섹션 검색
4. 권장 해결 방법 시도

### 만약 해결 안 되면
1. RISK_MATRIX에서 **관련 리스크** 찾기
2. **롤백 가이드** 참조
3. 이전 상태로 **복구**
4. 외부 지원 고려
