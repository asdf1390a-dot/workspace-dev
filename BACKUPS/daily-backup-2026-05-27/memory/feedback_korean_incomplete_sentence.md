## 문제 정의

**사건:** msg#5550 비서가 사용자의 불완성 한국어 문장("이거는")을 인식하지 못하고 "문장을 다시 말씀해주세요"라고 성급하게 응답.

**근본 원인 5가지:**
1. **LLM 통사 구조 한계** — 한국어 SOV 구조에서 술어 누락 시 "발화 종료"로 판단
2. **메시지 큐잉 부재** — Telegram 모바일에서 짧게 끊어 보낸 메시지를 즉시 처리
3. **SOUL.md 규칙 부재** — 불완성 처리 가이드라인 없음
4. **한국어 형태소 분석 불가** — 어미/조사 구분 도구 없음
5. **사용자 입력 패턴 인지 부족** — "타이핑 중" 신호 활용 불가

### 처리 규칙

| 패턴 | 예시 | 동작 |
|------|------|------|
| 조사로 끝남 | "이거는", "회사에서" | NO_REPLY (침묵) |

| 질문형 | "이게 뭐야?" | 즉시 응답 |
