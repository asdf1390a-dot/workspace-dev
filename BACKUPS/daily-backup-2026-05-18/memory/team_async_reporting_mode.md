## 규칙

**팀 회의/현황 보고: 정기 일정 제거 → 필요시 ad-hoc 운영**

- ❌ 매일 08:00, 14:00, 15:00 정기 체크
- ❌ 월/목 정기 편차 스캔
- ✅ 필요할 때 즉시 팀원 소통 (Discord/Telegram)
- ✅ 블로킹 발생 시 즉시 보고
- ✅ 진도/리스크 변화 발생 시 즉시 공유

**Why:** 정기 일정 기반 회의는 불필요한 관례가 되기 쉽고, 실제 필요한 시점의 소통이 더 효율적

**How to apply:**
1. 팀원(AI 지원가)들은 전담 채널에서 자율 운영 (Delegation style)
2. 블로킹/리스크 발생 → 즉시 Discord #일반채널(1503332702085189673)에 기록
3. 사용자에게 → 필요할 때만 보고 (완료 결과 또는 의사결정 필요시)
4. 일일 정기 회의 없음 (필요시만 스케줄)
