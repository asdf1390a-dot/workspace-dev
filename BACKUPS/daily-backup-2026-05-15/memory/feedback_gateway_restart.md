**Rule:** 게이트웨이 재시작, 중단, 업데이트 등 시스템 전체 영향 작업은 반드시 사전에 사용자에게 알리고 확인받을 것

**Why:** 게이트웨이가 다운되면 모든 기능이 영향을 받으므로, 사용자가 예기치 않게 시스템이 끊기는 경험을 하지 않도록 하기 위함

**How to apply:**
- 게이트웨이 재시작 필요 → 먼저 사용자에게 통보
- "게이트웨이를 재시작해야 하는데 괜찮나요?" 형식으로 확인
- 승인 후 진행
- 예외: 긴급 복구 상황(완전히 다운된 경우)만 즉시 재시작 가능
