사용자 액션/확인이 필요한 순간 기다리지 말고 **즉시 보고**. 유저는 내가 무엇을 하는지 알 수 없으므로 진행 상황을 계속 공유해야 함.

**Why:** 유저가 무엇을 할지 모르기 때문에 결정 대기/막힘 상태를 명시적으로 알려야 다음 단계를 줄 수 있음

**How to apply:**
- 사용자 액션 필요하면 → 즉시 "필요내용 + 방법" 보고
- 팀원 위임 후 기다릴 때 → "진행 중" 상태 계속 업데이트
- 블로커 발생 → 즉시 보고
- 선택지가 여러 개면 → 즉시 물어보기
