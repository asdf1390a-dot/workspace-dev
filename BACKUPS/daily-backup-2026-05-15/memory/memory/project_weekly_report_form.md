## 주간업무양식 (Weekly Report Form)

**기본 정보**
- 형식: Excel 스프레드시트
- 단위: 주차별 (월요일~일요일)
- 작성자: 부서장
- 제출: 매주 화요일 오전

## 구성 (5개 섹션)

### 1. 헤더
- 공장명 (만누르 공장)
- 작성자 정보
- 기간 (19주차 = 2026-05-01 ~ 2026-05-07)
- 작성일

### 2. 부서별 지표 (4개 부서)
각 부서마다 동일한 열 구조:

**생산부** (생산/QA/검사/부품 등)
- 항목명 (예: 생산량, 최종검사, 직렬검사, 부품검사 등)
- 계획 (Plan)
- 결과 (Result)
- 달성율 (%)
- 비고 (Notes)

**기술부** (동장비/공구/금형 등)
- 항목명
- 계획/결과/달성율/비고 동일 구조

**보전부** (전기/기계 등)
- 항목명
- 계획/결과/달성율/비고 동일 구조

**생산관리** (계획/품질/비용/기타)
- 항목명
- 계획/결과/달성율/비고 동일 구조

### 3. 성과지표 (우측 요약)
각 부서의 핵심 KPI를 백분율(%)로 표시:
- 예: 장비이용률 1.731%, 효율 1,600, 목표달성 등

### 4. 주요사항/이슈 (중단)
**우측:** 실천사항 (진행 중/완료)
**좌측:** 향후 계획

### 5. 기술획득 현황 (하단)
모든 개선/기술 항목 리스트업:
- 완료 현황 (✓/진행중/계획)
- 우선순위 (H/M/L)
- 마감일
- 담당자

**예시 항목:**
- 사출 금형 개선
- CCS 효율 업계 표준화
- BEST PRACTICE 실천
- EPC 추가설비 도입 계획 등 (최대 20~30개)

## 데이터 입력 규칙

| 항목 | 형식 | 단위 | 입력 방식 |
|------|------|------|---------|
| 계획/결과 | 숫자 | 건/개/시간 | 직접 입력 |
| 달성율 | % | 100% | = 결과/계획 (자동 계산) |
| 목표 달성 | 0~100% | % | 입력 또는 자동 |
| 개선사항 | 텍스트 | - | 자유 기술 |

## 특이사항
- 색상 구분: 생산(노랑), 기술(파랑), 보전(초록), 생산관리(회색)
- 목표 미달성시 주홍색 하이라이트
- 주간 비교: 전주 대비 추이 표기 (↑/↓)

## 작성 주기
- **매주**: 월요일~금요일 중 수집
- **제출**: 화요일 오전 (다음주 계획과 함께)
- **아카이빙**: 연간 52주치 누적
