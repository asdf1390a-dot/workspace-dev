사용자가 지시·선호·규칙을 말하면 **즉시 영구 저장**. 따로 "저장해줘" 요청 없이도.

**Why:** 유저가 명시적으로 "내가 시킨 건 판단해서 스킬/SOUL.md에 영구저장해, 잊어버리지 말고"라고 요청. 대화가 끊기면 채팅 기록은 사라지므로, 기억할 가치가 있는 모든 지시는 파일로 보존해야 함.

**How to apply:**
- 저장 위치 판단 기준:
  - `SOUL.md` → 나의 행동 방식, 말투, 작업 원칙
  - `memory/*.md` → 프로젝트 사실, 피드백, 사용자 프로파일, 레퍼런스
  - `skills/*.md` → 반복 가능한 절차·워크플로우
- 지시를 받는 즉시 저장. 작업 끝난 후로 미루지 말 것.
- 저장했음을 별도로 고지할 필요 없음 — 그냥 하고 넘어갈 것.
