**규칙:** 테이블 및 목록 UI의 색상 대비를 명확하게 개선 필요 (헤더, 데이터 셀 모두)

**Why:** 사용자가 자산 목록 페이지에서 일부 항목이 회색으로 표시되어 안 보인다고 지적. UI 컴포넌트의 색상이 배경과 충분히 대비되지 않으면 정보 접근성이 낮아짐.

**How to apply:** 
- 페이지네이션 수정(Step 1) 또는 BM 모듈 UI(Step 2&3) 구현 시 테이블/리스트 스타일링 개선
- 헤더 배경색: 명확한 파란색/회색 계열 (밝은 톤)
- 데이터 텍스트: 검정색 또는 어두운 회색 (백그라운드와 충분한 명도 차이)
- 비활성 항목도 구분 가능한 색상 유지
- WCAG AA 표준 이상의 명도 비율(4.5:1) 권장
