번역 작업 표준 순서:
1. 번역가가 자연스럽게 번역 진행 (컨펌 없이)
2. 평가자가 별도로 품질 평가 (최소 3회 반복 검증 기준)
3. 평가 완료 후 결과 보고 (번역본 + 평가 의견 함께)

**Why:** 번역 품질 보장을 위해 번역가와 평가자 역할 분리 명시적 지시.

**How to apply:** 번역 요청 들어오면 번역가 서브에이전트 먼저 실행, 완료 후 평가자 서브에이전트에게 검토 위임, 둘 다 완료 후 최종 보고.
