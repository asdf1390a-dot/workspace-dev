## 단위 체계 (시트별)

| 시트 | 단위 |
|------|------|
| `월계획 `, `경비`, `매출매입` | 천원 (KRW ÷ 1,000) |
| `원가절감`, `클레임&폐기비` | 백만원 (KRW ÷ 1,000,000) |
| `클레임&폐기비` 매출액 행 | 백만원 (INR × 15.5 ÷ 1,000,000) |
| `IS`, `IS (Tally)`, `재무상태표(BS)` | INR (Tally 원본) |
| `불량율` | 건수 + PPM |

## 직접 입력이 필요한 시트 (Korea Report 기반)

| 시트 | 입력 내용 |
|------|-----------|
| `불량율` | 고객불량/공정불량/입고불량 건수·PPM 월별 |
| `클레임&폐기비` | 클레임비·폐기비 월별 (백만원), 매출액 (INR→KRW 환산) |
| `월지표` | 월별 KPI 지표 전체 |
| `IS`, `IS (Tally)` | Tally ERP 원본 데이터 (재무팀 담당) |
| `월계획 ` | 매출·원가 월별 실적 (재무팀 담당) |

## PT 시트 (수식 자동 계산 — 직접 입력 금지)
`3.경영실적 PT`, `경영실적 추이도 PT`, `경영실적 분석 PT 1/2`, `성과지표 분석 PT 1`
→ 위 시트들은 수식으로 자동 계산. 절대 건드리지 말 것.

## 핵심 주의사항
- **openpyxl 재저장 금지**: 1.8MB → 195KB로 축소되며 차트/이미지/VBA 파괴됨
- **수정 방법**: 파일 복사 후 ZIP/XML 직접 수정, 또는 원본 복사본 유지
- **빈칸 정책**: 발생 없는 항목은 0이 아닌 blank 유지
- **매출액 단위**: `월계획 ` 시트는 천원 단위 (7,468백만원 = 7,468,000 입력)
