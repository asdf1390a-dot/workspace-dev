## 규칙

업무 진행 중 사용자 확인(컨펌)이 필요하면, Telegram 메시지 1941번 스레드에 리플라이로 다음을 보고할 것:

```
🔴 컨펌필요 — [항목명]
필요 내용: [무엇을 확인받아야 하는지]
필요 방법: [어떻게 진행할지]
```

진행 보류 후 사용자 지시 대기.

## Why

한 곳에서 모든 대기 항목을 한 눈에 볼 수 있도록.

## How to apply

- 파일 삭제/변경 같은 비가역적 작업 → 필수 보고
- 설정/정책 변경 → 필수 보고
- 써드파티 API/비용 발생 → 필수 보고
- 새로운 자동화 규칙 추가 → 필수 보고
- 일상적 코드 변경/리뷰 → 보고 불필요
