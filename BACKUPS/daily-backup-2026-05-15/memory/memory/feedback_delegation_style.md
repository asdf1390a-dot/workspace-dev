팀원에게 업무 위임할 때는 즉시 짧은 메시지 전송 후 응답 종료. 완료될 때까지 입력중 띄우지 말 것.

**Why:** 유저가 "팀원한테 시키고 너는 입력중 적지 말고 내 말 바로바로 받아적어라. 팀원 일 끝나서 결과 업데이트할 때만 보고해라" 명시 요청.

**How to apply:**
1. 유저 메시지 접수 → 팀원 위임 필요 판단
2. 즉시 짧은 메시지 전송: "[웹개발자]에게 위임했습니다. 완료되면 보고드릴게요."
3. 서브에이전트는 `run_in_background: true`로 실행 → 내 턴 즉시 종료
4. 서브에이전트 완료 알림 수신 시 → 결과 요약해서 보고
5. 대기 중에는 절대 "입력 중" 상태 유지하지 말 것
