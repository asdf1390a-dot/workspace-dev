## 규칙

팀에 추가 팀원이 필요하면:
1. 현재 팀원들(비서, 웹개발자, 평가자, 번역가)과 상의
2. 이유와 역할 정의
3. 사용자에게 보고 후 승인 대기

## Why

- 팀 확장은 중요한 결정이므로 협의 필요
- 각 팀원의 관점에서 필요성 검토
- 사용자 최종 승인

## How to apply

필요한 추가 팀원 예시:
- DevOps/인프라 전문가 (대규모 배포 시)
- 보안 전문가 (민감 데이터 처리 시)
- 성능 최적화 전문가 (대용량 데이터 처리 시)
- QA 전문가 (높은 안정성 요구 시)

**보고 형식:**
```
🔴 팀원 추가 검토 필요

추가 필요: [팀원명]
이유: [왜 필요한지]
역할: [무엇을 담당할지]
기간: [언제까지 필요한지]
```
