## 6️⃣ 독립성 검증 기준 (Phase B 독립 작업 가능 여부)

**독립성 7가지 신호:**
1. ✅ **자문 없는 문제 해결:** 멘토 없이 기술 이슈 해결 (1시간 내)
2. ✅ **자율적 우선순위 결정:** 작업 순서를 스스로 정함
3. ✅ **명확한 커뮤니케이션:** 진도를 자발적으로 보고 (≥3회)
4. ✅ **품질 기준 이해:** 합격선(95% 정확도) 스스로 적용
5. ✅ **시간 관리:** 예정된 시간에 산출물 제출 (3/3회)
6. ✅ **문서화 자주성:** 이해도 문서/메모 자발적 작성
7. ✅ **멘토 피드백 적용:** 지시사항 빠르게 반영 (<2시간)

**판정:**
- 7/7 충족 → 🟢 **완전 독립** (Phase B 단독 프로젝트 진행 가능)
- 5-6/7 충족 → 🟡 **부분 독립** (Phase B 진행 + 약한 영역 모니터링)
- <5/7 충족 → 🔴 **미성숙** (Phase A 연장 필요)
