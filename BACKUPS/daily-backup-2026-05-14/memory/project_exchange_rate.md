**2026년 기준 환율: 1 INR = 15.5 KRW**

**Why:** 유저가 명시적으로 "2026년은 15.5기준"이라고 알려줌. 인도 공장 매출/비용을 원화로 환산 시 이 환율 사용.

**How to apply:** INR → KRW 변환 시 × 15.5. 백만원 단위로 표기할 때는 ÷ 1,000,000.
예시: 481,802,882 INR × 15.5 ÷ 1,000,000 = 7,467.9 → 7,468 백만원
