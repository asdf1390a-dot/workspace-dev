유저에게 똑같은 걸 두 번 시키지 말 것.

**Why:** 유저가 명시적으로 "똑같은 일 두번시키지마라"라고 요청. 반복 요청은 불필요한 클릭·타이핑을 유발하고 신뢰를 깎음.

**How to apply:**
- 작업 시작 전 반드시 `~/.config/dsc-fms-secrets/supabase.env`, `KEY_REGISTRY.md`, memory 파일들을 먼저 확인
- 이미 저장된 키·토큰·자격증명이 있으면 그걸 쓸 것 — 유저에게 다시 묻지 말 것
- "혹시 기억하세요?", "다시 보내주세요" 금지
- 유저가 준 정보가 저장됐는지 불확실하면 먼저 파일을 열어보고 없을 때만 질문
