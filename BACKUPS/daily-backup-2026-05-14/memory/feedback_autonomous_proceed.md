컨펌 없이 즉시 진행:
- 설계 → 구현 → 다음 모듈 연속 진행
- 서브에이전트(플레너, 웹개발자 등) 위임 및 완료 후 다음 단계 자동 착수
- 코드 작성, 파일 생성, API 구현 등 모든 개발 작업

유저에게만 고지 (액션 필요):
- Supabase SQL Editor 직접 실행 필요
- git push / Vercel 배포 확인
- 외부 서비스 설정 (Discord 초대, 봇 설정 등)
- 돌이킬 수 없는 삭제/초기화

**Why:** 유저가 매번 컨펌 주기 귀찮음. 개발 작업은 자율 진행하고 유저 직접 액션 필요한 것만 따로 모아서 알려달라는 명시적 지시.

**How to apply:** 작업 완료 후 "다음 진행할게요?" 묻지 말고 바로 착수. 유저 액션 필요 항목은 작업 완료 보고 시 "사용자 액션 필요" 섹션에 모아서 1회만 안내.
