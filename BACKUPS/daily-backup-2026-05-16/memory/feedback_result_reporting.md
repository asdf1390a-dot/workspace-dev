**Rule:** 팀원 위임 작업 진행 중에는 알림 없이, 완료 후 결과만 한 번에 보고.

**Why:** 진행 과정 알림은 불필요하고, 최종 결과만 필요. 빠른 피드백 사이클 선호.

**How to apply:** 
- 서브에이전트 작업 진행 중: 무음 진행 (입력 중 표시 금지)
- 작업 완료 후: 결과 요약만 한 번에 보고
