## 규칙

**사용자 액션 필요 항목은 항상 "현황표" 형식으로 정리**

## 표준 형식

```
## 【사용자 액션 필요】

### 🔴 즉시 필요 (N개)

#### 1️⃣ **[액션명]**
**기한:** [날짜]  
**소요시간:** [시간]

**📍 접속:**  
[클릭가능한 사이트 URL]

**📄 파일/코드:**  
[다운로드 또는 복사할 명령어]

**⚙️ 실행 방법:**
1. [단계 1]
2. [단계 2]
3. [단계 3]

---

#### 2️⃣ **[다른 액션명]**
...

---

### 🟡 이번주 (N개)
...

---

**완료 후 각 항목 옆에 ✅ 체크해주세요.**
```

## Why

- 사용자가 여러 작업 중 액션이 필요할 때, 명확하게 "무엇을 언제 어떻게" 실행할지 한눈에 파악 필요
- 링크·명령어·단계별 방법을 함께 제시해야 반복 요청 없음
- 표로 정렬하면 우선순위 구분 명확 (🔴즉시/🟡이번주/🔵다음주)

## How to apply

1. 사용자 액션 필요할 때마다 **이 형식으로 정리**
2. 📍접속(URL) + 📄파일(링크/명령어) + ⚙️단계(3단계 이내) **모두 포함**
3. 🔴(즉시) / 🟡(이번주) / 🔵(다음주)로 **우선순위 명확히**
4. 메시지는 1회만 → 같은 항목 반복 설명 금지
