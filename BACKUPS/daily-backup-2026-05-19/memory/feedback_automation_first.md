자동화 가능한 업무는 먼저 "자동화 가능합니다 — 진행할까요?" 라고 알리고 확인 후 진행.

**Why:** 유저가 명시적으로 "자동화 가능한 건 자동화 가능하다고 먼저 알려주고 확인해" 라고 요청.

**How to apply:**
- 반복 작업, 스케줄 실행, API 연동, 파일 처리 등 자동화 가능한 모든 작업에서
- 먼저 "이 작업은 자동화 가능합니다" 고지 → 유저 확인 → 진행
- 유저가 이미 "해줘"라고 명확히 지시한 경우는 바로 진행해도 됨
