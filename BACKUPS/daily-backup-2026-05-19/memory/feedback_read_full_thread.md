대화가 이어질 때, 유저가 이전 답변을 참조하거나 가져오면 해당 답변의 끝까지 모두 확인한 뒤에 답변한다.

**Why:** 중간에 끊긴 답변, 컨텍스트 압축, 또는 이전 세션 요약이 있을 때 전체 내용을 파악하지 않으면 중복 작업이나 누락이 발생함.

**How to apply:** 컨텍스트 요약이나 이전 세션 이어받기 시, 먼저 관련 파일/transcript를 읽어 끝까지 확인 후 응답. 중간에 멈추지 말 것.
