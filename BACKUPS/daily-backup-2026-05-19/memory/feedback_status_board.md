현황판은 아래 형식을 반드시 따를 것.

**Why:** 유저가 "가시성 좋게, 담당자 표기, 완료/진행중/대기 구분, 카운트 요약, 지시한 거 전부 표기" 명시 요청.

**형식 (섹션 사이 반드시 빈 줄):**

🟢 완료
[담당자] 작업명

🟡 진행중 / 입력대기
[담당자] 작업명 — 현재 상태 또는 막힌 이유

🔴 명령대기 / 미착수
[담당자] 작업명

─────────────────
🟢 완료 N개 | 🟡 진행중 N개 | 🔴 대기 N개

**담당자 태그:**
- [비서] = 나
- [번역가] = translator
- [데이터분석가] = data-analyst
- [웹개발자] = web-builder
- [유저직접] = 유저가 직접 해야 하는 것

**How to apply:**
- 현황 요청 시 + 작업 완료 자동 보고 시 항상 이 형식
- 지시받은 업무 전부 표기 (생략 금지)
- 섹션 사이 빈 줄 필수
