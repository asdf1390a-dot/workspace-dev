## 규칙

**유저 액션 필요 항목은 "사용자 액션 필요" 섹션에 모아서 1회만 안내**

## 구조

```
✅ [작업 완료 설명]

---

## 사용자 액션 필요

### 액션 1: [액션명]
- **방법 1:** [방법1 상세]
- **방법 2:** [방법2 상세] ← 추천
- **직접 링크:** [클릭가능한 URL]

### 액션 2: [다른 액션]
...

---

완료 후 ✅ 알려주세요
```

## Why

- 유저가 반복되는 요청에 불평함 (2026-05-15 00:27 KST)
- 규칙 위반으로 신뢰도 하락
- 개발자는 한 번에 정보를 받아야 함 (ADHD/인지 부하)

## How to apply

1. 유저 액션 필요 시, 먼저 **모든 방법을 준비**
2. 한 메시지로 **모든 옵션을 정리된 형태로 제시**
3. 다시 절대 같은 항목으로 메시지 보내지 않음
4. 메모리에 적혀있지 않으면 **이 형식 사용 불가**

## 금지 사항 ❌

- 액션 필요 항목을 여러 메시지로 나누기
- 같은 유저 액션을 반복 설명
- 한 번에 한 가지만 제시하고 나머지는 나중에 (한 번에 다)
