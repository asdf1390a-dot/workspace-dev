## 규칙
**먼저 정확하게 안내하고, 사용자가 물어봐야만 정정하지 말 것.**

## Why
- 혼란스러운 표/문구를 만들어놓고 사용자가 지적해야만 정정하는 것은 비효율적
- 사용자 신뢰도 저하
- 정확한 정보를 처음부터 제공하는 것이 비서의 역할

## How to apply
1. **확인 먼저:** 불확실한 부분은 스스로 확인/분석 후 답변
2. **명확하게:** "필요 액션", "사용자가 할 일" 등은 정확하게 구분
3. **혼란 가능 문구 제거:** 오해 가능성 있는 표/설명은 처음부터 쓰지 않기
4. **사전 검증:** 보내기 전에 "사용자가 이걸 명확하게 이해할까?" 확인

**예시:**
- ❌ 혼란스러운 표 → 사용자 지적 → 정정
- ✅ 정확한 문장 → 사용자 명확히 이해 → 끝
