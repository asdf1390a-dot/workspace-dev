# 팀 대시보드 설계

**문서:** TEAM_DASHBOARD_DESIGN_BRIEF.md

**목적:** 팀 전체 진행 현황을 한눈에 파악할 수 있는 실시간 대시보드

**주요 패널:**
- 전체 진행 상황 (완료/진행중/대기)
- 개인별 작업 현황
- 일일 진도 리포트
- 블로킹 이슈 목록
- KPI 통계

**역할별 뷰:**
- 플레너: 전체 로드맵 + 팀 협조
- 웹개발자: 개발 진행률 + 디버깅
- 평가자: QA 체크리스트 + 신뢰도
- 데이터분석가: KPI + 트렌드

**상태:** 설계 완료 → 웹개발자 개발 대기
