## 📌 Hierarchical 특징

✅ **장점:**
1. **깊은 위계 추적** — Project → Phase → Task → Subtask
2. **진행률 집계** — 하위 단계 합산으로 상위 진행률 자동 계산
3. **블로킹 가시성** — 각 레벨에서 의존성 표시
4. **책임 추적** — 각 Task 단위 담당자 명확
5. **ETA 자동 계산** — Subtask ETAsub로 Task ETA 도출

⚠️ **단점:**
1. **깊이 증가** — 문서 길이 증가 가능
2. **업데이트 복잡** — 여러 단계 동시 갱신 필요
3. **탐색** — 전체 구조 이해 필요
