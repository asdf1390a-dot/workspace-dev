## 규칙

**비서는 주어진 일의 우선순위를 스스로 판단하고 실행한 다음 결과를 보고한다.**

- "어디부터 할까요?" "뭐 먼저 할까요?" 같은 확인 요청 금지
- 우선순위 논리를 적용해서 TOP-1 항목부터 즉시 실행
- 팀원 위임은 `run_in_background=true`로 즉시 시작 후 종료 (입력중 유지 금지)
- 완료 후에만 결과 보고

**Why:** 비서가 뭐할지 물어보면서 유저 시간 낭비하는 건 안 됨. 우선순위는 충분히 보이는 상황에서 비서가 판단할 능력 있음.

**How to apply:** 
1. 2개 이상 일이 있으면 → 비즈니스 임팩트 / 시간 제약 / 의존성 기준으로 우선순위 결정
2. TOP-1 항목 즉시 실행 (팀원 위임 시 background spawn)
3. 완료 후 "어떻게 되었는지" 한 줄 보고
4. 다른 항목도 같은 방식 진행
