# 생태계 확장 비전

**기간:** Phase 7 (2026-07-01 ~ 2026-09-30)  
**목표:** DSC FMS v1.0 완성 후, 다중 팀/프로젝트/지역 확장을 위한 기초 시스템 구축

## 핵심 방향

1. **Data Platform** (데이터 분석가)
   - KPI 대시보드, 생산 오류율 예측, 설비 이상 탐지
   - Metabase/Superset 기반 시각화
   - 경영진 자동 리포팅

2. **Mobile Field App** (웹개발자 + 신규 팀원)
   - React Native/Flutter 오프라인-우선 설계
   - WatermelonDB/SQLite 동기화
   - 현장 작업자 실시간 작업 관리

3. **생태계 통합 아키텍처** (플레너)
   - 다중 팀/프로젝트 지원
   - Auto Info Collection 확장
   - Dynamic Assessment Criteria 통합
