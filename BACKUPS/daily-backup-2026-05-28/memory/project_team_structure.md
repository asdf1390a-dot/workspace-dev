## 절대 규칙

**팀원 = AI 직원들 (시뮬레이션)**
- 평가자 (Evaluator AI)
- 플레너 (Planner AI)  
- 웹개발자 (Web Developer AI)
- 데이터분석가 (Data Analyst AI)
- 번역가 (Translator AI)

**현실 팀원 = 없음**
- 사용자가 명시적으로 추가 지시 전까지
- 이 대화에서는 사용자(나)와 비서(나) 두 명만 존재
- 실제 사람과의 Telegram/Discord/메일 연락 불가

## How to Apply

**팀 체크인/회의 시뮬레이션:**
- 직접 각 AI 팀원 관점에서 의견/피드백 생성
- 사용자에게 "팀이 뭐라고 생각할까" 물어보지 말 것 (팀이 없으므로)
- 상황과 맥락에 따라 각 AI 팀원의 입장에서 합리적 피드백 구성

**메시지/알림:**
- Discord, Telegram, Slack 등 외부 채널에 팀원 메시지 금지
- 모든 팀 커뮤니케이션은 이 채팅에서만 진행

**Future Rule:**
- 사용자가 "실제 팀원 추가" 명시할 때만 현실 인수인계 가능
- 그때도 추가 동의 및 규칙 업데이트 필요
