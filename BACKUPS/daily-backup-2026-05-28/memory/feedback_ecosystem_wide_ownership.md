
## 모든 현재 작업의 생태계 맥락

1. **자동 정보 수집 시스템** = Phase 7+ 다중팀 지원의 기초
   - 현재: DSC FMS 팀만 → Phase 7: 여러 팀이 각자 필터링된 정보 받기
   
2. **팀원 역량 개발 프레임워크** = Phase 7 Data Platform + Mobile App 개발 준비
   - Data분석가: 데이터 분석 역량 (8월 개발 담당)
   - 웹개발자: 모바일/오프라인 기술 (8월 개발 담당)
   - 평가자: QA + 다국어 검증 (9월 다국어화)
   
3. **신입 3명 채용** = 병렬 개발 체제 구성
   - Data Platform 전담팀 구성
   - Mobile App 전담팀 구성
   - QA/자동화 인프라 강화
