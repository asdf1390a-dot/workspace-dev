**규칙:** 세션을 장시간 유지하면서 작업의 맥락과 흐름을 보존하라.

**Why:** 
- 각 턴마다 ScheduleWakeup으로 세션 종료 시 속도는 빨아지지만
- 매번 컨텍스트 손실 → 작업 기억 불가 → 미묘한 상태 변화를 놓침
- "멍청해진" 상태는 이전 진행상황 + 중간 판단을 잃고 있기 때문

**How to apply:**
1. 세션을 장시간 열어두고 내부에서 작업 순차 관리
2. ScheduleWakeup보다 작업 흐름 내에서 자연스러운 진행
3. 작업 중 상태 변화는 기억(메모리) + 작업 컨텍스트로 함께 추적
4. 필요시에만 세션 분리 (정말 필요한 경우)

이렇게 하면 속도도 빠르면서 정확성(작업 기억력)도 유지됨.
