## 규칙 (2026-05-19 수립)
사진이든 영상 편집 요청 받으면 → **Google Drive 공유링크로 제공** → 사용자가 자가편집.
비서는 편집본을 전달하지 않고, 링크만 제공.

## 배경
- 사용자가 휴가 중(2026-05-15~24) 폰만 사용 가능
- 폰으로 파일 다운로드 불가, 클라우드 편집 선호
- Google Drive에서 직접 편집 및 공유 가능

## 처리 프로세스
1. **사진/영상 요청 받음** — 편집 내용 파악
2. **Google Drive 업로드** — 원본 또는 기초 편집본 업로드
3. **공유링크 생성** — 수정 가능한 링크 제공
4. **사용자 자가편집** — 폰에서 Drive 앱으로 편집

## 적용 사례 (2026-05-19)
- Cartoon Filter 적용 영상 (237MB)
- Telegram 전송 불가 (파일 크기)
- OpenCV 압축: 237MB → 16.6MB (93% 감소, 480x270, 15fps)
- Telegram 전송 완료 (messageId: 5078)

## 왜
사용자가 폰 환경에서 클라우드 편집을 원함. 비서가 편집본 전달 → 폰 다운로드 불가 → 비효율.

## 확장 범위
- **사진:** 필터, 색상 조정, 배경 제거, 워터마크, 스타일 변환 등 (Google Drive에서 가능한 것)
- **영상:** 자막, 효과, 순서 조정 등 (Google Photos/Drive 앱 활용)
- **제약:** Drive에서 불가능한 고도 편집은 사전 상담

## 적용 방법
사진/영상 편집 요청 → 원본/기초본 Drive 업로드 → **Google Drive 공유링크 제공만**. 편집본 제공 금지.
