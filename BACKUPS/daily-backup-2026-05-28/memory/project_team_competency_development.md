# 팀 역량 개발 프레임워크

**목적:** 팀 전체가 항상 더 똑똑해지고 발전하도록 지속적 학습 + 역량 강화  
**운영:** 비서 주도 + 월간 팀 검토 회의

## 4개 모듈

### Module 1: 자동 정보 수집 시스템
- GitHub Trending, Product Hunt, Dev.to 자동 수집
- AI 필터링 (역할별 관련도)
- Telegram으로 매일 배포

**상세:** TEAM_DISCUSSION_AUTO_INFO_COLLECTION_SYSTEM.md

### Module 2: 팀 기술 스택 교육
- 각 도메인별 최신 기술 트렌드
- 경쟁사/업계 리더 벤치마크
- 월간 스터디 세션

**담당:** 데이터분석가 (기술 속도 분석)

### Module 3: 역할별 로드맵
- 웹개발자: React Native/Flutter, Mobile 오프라인 동기화
- 데이터분석가: Python 예측 모델, 이상 탐지
- 번역가: 다국어 현지화 (KO/EN/HI)

**기한:** 2026-06-30까지 완성

### Module 4: 피드백 루프
- 월간 팀원 의견 수렴
- 성과 평가 기준 동적 업데이트
- 학습 효과 측정

**상세:** ASSESSMENT_CRITERIA_DYNAMIC_SYSTEM.md

## 성공 지표

- 팀원 당 월 3회 이상 새로운 기술 습득
- 정기 성과 기준 달성률 95% 이상
- 팀 전체 기술 깊이 + 폭 매 분기 증가
