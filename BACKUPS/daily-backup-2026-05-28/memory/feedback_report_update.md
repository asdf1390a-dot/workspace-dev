월간 보고서(2번+3번 파일) 업데이트 시 전월 완성본과 동일한 패턴으로만 작업할 것.

**Why:** 유저가 명시적으로 "같은 방식으로, 더하려고도 하지 말고 덜하지도 말고"라고 요청. 없는 데이터(매출액, M/H 등)는 타 부서 담당이므로 빈칸으로 두는 것이 맞음.

**How to apply:**
1. 전월 완성본 vs 전전월 완성본 비교 → 변경된 셀 목록 파악
2. 해당 변경 패턴을 Korea Report 기준으로 그대로 적용
3. Korea Report에 없는 데이터는 건너뜀 (경고 메시지 최소화)
4. 수식 셀 절대 건드리지 않음
5. **사소한 변경도 빠짐없이 수정** — 작다고 건너뛰지 말 것
6. **폰트 건드리지 말 것** — 글꼴·크기·색상 등 서식 일절 변경 금지
7. **표 내용이 길어지면 자동 조정** — 행 높이·열 너비·텍스트 줄바꿈 등을 알아서 조정해 넘침 없이 보기 좋게 만들 것 (레이아웃 변경은 허용, 폰트 변경은 금지)
