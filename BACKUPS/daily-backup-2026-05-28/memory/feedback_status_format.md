현황 보고는 반드시 아래 양식 그대로 사용. 임의로 바꾸지 말 것.

```
🟢 완료
[담당자] 내용

🟡 진행중
[담당자] 내용 — 사유

🟠 명령대기 (유저 액션 필요)
[유저직접] 무엇을 해야 하는지 상세 설명
→ 링크: https://...
→ 방법: 단계별 안내

🔴 미착수
[담당자] 내용

─────────────────
🟢 완료 N개 | 🟡 진행중 N개 | 🟠 대기 N개 | 🔴 미착수 N개
```

**색상 기준:**
- 🟢 완료
- 🟡 진행중 (AI 작업 중)
- 🟠 명령대기 — 유저가 직접 해야 하는 것. 상세 설명 + 링크 필수
- 🔴 미착수 — 아직 시작 안 한 것

**담당자 태그 (필수 — 모든 항목에 반드시 표기):** [비서] [웹개발자] [플레너] [번역가] [데이터분석가] [평가자] [유저직접]
→ 담당자 없는 항목 금지. 모든 줄에 [태그] 포함.

**Why:** 유저가 직접 양식 지정. 주황색(🟠)은 유저 액션 필요 항목 — 상세 내용과 링크 포함해야 모바일에서 바로 실행 가능.

**How to apply:** /현황 요청 시 또는 작업 완료/중단 시 자동 출력. 유저 액션 항목은 절대 간략하게 쓰지 말 것.
