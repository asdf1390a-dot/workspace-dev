**절대 규칙:**
- 이 양식 변경 금지 (사용자가 변경하라고 할 때까지)
- 항상 이 형식 사용
- 영혼에 새기고 절대 벗어나지 말 것

**양식:**

```
───

🟢 완료
[담당자] 작업 내용
[담당자] 작업 내용

🟡 진행중 / 입력대기
[담당자/상태] 작업 내용
[담당자/상태] 작업 내용

🔴 명령대기 / 미착수
[담당자] 작업 내용
[담당자] 작업 내용

─────────────────
🟢 완료 X개 | 🟡 진행중 X개 | 🔴 대기 X개
```

**명령대기 항목 기재 규칙:**
- 사용자 명령이 필요할 때만 포함
- **필요한 내용**과 **수행 방법** 정확히 기재
- 예시:
  ```
  [유저직접] Discord 봇 토큰 발급 — discord.com/developers 에서 봇 생성 필요
  [비서] 설정 추가 — DISCORD_WEBHOOK_URL=<토큰> to .env.local
  ```

**변경 조건:**
- 사용자가 명시적으로 "양식 변경하자" 또는 "다시 정하자" 할 때만
- 그 외 절대 변경 불가

**사용 시기:**
- 모든 현황 보고/진행상황 업데이트 시
- 프로젝트 단위 작업 현황
- 멀티태스크 추적
