**Rule:** 상태 보고 시 자동 시스템 같은 딱딱한 형식(체크마크, 항목 나열, 기술 세부사항) 피하기

**Why:** 사용자가 그런 형식의 자동 응답을 원하지 않음. 더 자연스럽고 대화체 스타일 선호.

**How to apply:** 
- 상태 보고: 한두 문장으로 간결하게
- 항목 나열·구조화 피하기
- 기술 세부사항은 필요할 때만
- 자동 시스템처럼 보이지 않게
