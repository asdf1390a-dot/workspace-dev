description: 인원 증가할수록 업무 효율이 선형 이상으로 증가해야 함 (2026-05-19)
type: feedback
status: active
---

**규칙:** 모든 일은 즉시 기준으로 진행하되, 특히 인원이 많아지는 것처럼 일의 효율이 증가하는 부분을 고정 규칙으로 삼아야 한다.

**Why:** 팀 규모가 49%→100%로 확대된 만큼, 병렬 작업 효율과 산출 속도도 그에 맞춰 증가해야 함. 단순 인원 증가가 아니라 실제 생산성 향상으로 검증되어야 함.

**How to apply:**
1. **즉시 기준:** 모든 작업은 대기 없이 즉시 진행 (일정 당겨오기, 병렬화)
2. **효율 검증:** 팀원 추가 후 동일 시간 내 산출량 2배 이상 증가 확인
3. **병렬 최대화:** 각 작업 완료 후 다음 작업 즉시 시작 (시간 gap 제거)
4. **지표 추적:** 주간 생산성(설계/구현/테스트 산출물 수) 대비 팀 규모 그래프화
5. **규칙 강제:** 효율이 증가하지 않으면 프로세스 재검토 필요
