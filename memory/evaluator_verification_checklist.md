# 평가자 검증 필수 체크리스트

**규칙:** 모든 배포는 다음 3가지까지 검증한 후에만 "배포 완료" 기록

## 필수 검증 항목 (3단계)

### 1️⃣ 빌드 상태 검증
```bash
npm run build
# ✅ 성공 여부 확인 (페이지 수, 에러)
```

### 2️⃣ HTTP 라우트 검증 (NEW)
```bash
curl -I https://dsc-fms-portal.vercel.app/login
# ✅ HTTP 200 확인
# ✅ HTML 렌더링 확인 (title 등)
```

### 3️⃣ 기능 동작 검증
```
예: 로그인 페이지
- [ ] 이메일/비밀번호 입력 박스 렌더링
- [ ] 로그인 버튼 클릭 가능
- [ ] 에러 메시지 표시 여부
```

---

## 체크리스트 (검증 후에만 체크)

- [ ] 빌드: npm run build 성공
- [ ] 라우트: HTTP 200 + 페이지 로드
- [ ] 기능: 주요 기능 동작 확인
- [ ] 배포: Vercel 실시간 URL 접속 완료

**검증 미완료 상태:** "빌드 성공, 배포 진행 중" (검증 대기 중)  
**검증 완료 상태:** "배포 완료" (3가지 모두 확인)
