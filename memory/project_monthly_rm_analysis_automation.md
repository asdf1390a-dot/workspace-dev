## 지시

매월 같은방식으로 자료업데이트 → 비서 자동 처리

## 방식 (4월 기준)

1. **입력:** 월간 R&M 예산 데이터 (카테고리별 비용)
2. **분석:**
   - 카테고리별 비용 구성 (금액 + 비중 + 월평균)
   - 월별 추이 & 이상치 (전월 대비 변화율)
   - 🔴 긴급 조치 필요 항목 3가지 (장비 고장, 누적 비용 등)
3. **출력:** 한글 테이블 + 분석 보고서

## 실행 계획

- **언제:** 매월 마감 후 (월 1-5일 추정)
- **데이터:** 사용자 제공 (또는 SAP 자동 추출 검토 필요)
- **보고:** 월간 분석 결과 Telegram 보고

## How to apply

매월 4월과 동일한 형식으로 분석 & 보고. 데이터 수신 시 즉시 처리.

**참고:** 매월 구체적 실행 일정은 사용자 지시 대기
