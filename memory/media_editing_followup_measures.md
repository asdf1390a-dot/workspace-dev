**규칙:** 편집 요청 시 비서가 직접 처리, 경로·모델·필터 절대 묻기 금지.

**Why:** 2026-05-19 19:21에 사진/영상 편집 규칙 위반 — 비서가 경로 불명확하다고 사용자에게 물음 (자율 모드 위반).

**How to apply:**

### 즉시 조치 3가지
1. **SOUL.md에 명시** ✅ — "사진/영상 편집 자동 처리" 항목 추가 (경로만 받고 즉시 처리)
2. **자동화 체크리스트 추가** — 매일 audit log에 "편집 요청 시 경로 묻지 않았는가?" 확인
3. **Discord 팀 기록** — 이 사건과 해결책을 #일반채널에 기록 (팀 학습용)

### 향후 규칙
- **입력:** 사용자가 "영상 편집해줘" + 파일 경로 → 비서가 즉시 처리
- **금지:** 어떤 필터/모델인지, 어디 저장할지, 확인 필요 여부 절대 묻기
- **출력:** 처리 완료 파일 경로 제공
- **모호함 0화:** 이전에 사용한 필터/효과 기억해서 같은 방식으로 자동 적용

### 감시 지점
- **08:00 KST 정기 체크:** "자율 모드 규칙 위반 없었는가" 확인 항목 추가
- **월 1회 audit:** 편집 요청 → 처리까지의 타임라인 검토 (경로/모델 묻지 않았는가)
