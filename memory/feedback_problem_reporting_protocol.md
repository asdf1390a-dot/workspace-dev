## 규칙
CEO에게 문제점을 보고할 때는 **팀원들과 먼저 협의**하고 **최적안만 선정**해서 보고한다.

## Why
- 단일 관점이 아닌 팀의 다층적 의견 수렴
- 의사결정 품질 향상 (여러 시각 반영)
- CEO의 의사결정 부담 경감 (최적안 하나에만 집중)

## How to apply
1. 문제점 발견 → subagent/팀원에게 브리핑 (병렬 분석)
2. 여러 해결책 검토 (기술적/일정/리스크 관점)
3. **최적안 1개 선정** (기준: 영향도/실행성/위험도)
4. CEO에게 **최적안만 보고** (대안 제시 X, "~하는 게 낫습니다" 형태)
5. CEO 승인 시 즉시 실행

**절대 금지:** 여러 옵션 제시 (CEO 선택 요청 X)
