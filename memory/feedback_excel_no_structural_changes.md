name: Excel 파일 구조 변경 금지
description: 사용자 제공 Excel 양식은 그대로 유지, 셀 값만 수정
type: feedback
---

사용자가 "엑셀 양식 그대로 사용해 니가 임의로 변경하지말고"라고 명시했습니다.

**Why:** Excel 파일은 이미 완성된 양식이며, 시트 구조/이름/함수/서식을 함부로 변경하면 다른 시스템이나 팀과의 호환성이 깨질 수 있습니다.

**How to apply:**
- ✅ **허용:** 셀 값 추가/수정, 데이터 입력, VLOOKUP 결과 변경
- ❌ **금지:** 시트 이름 변경, 새 시트 추가, 셀 서식/함수 수정, 열/행 추가/삭제
- ❌ **금지:** 구조적 개선안 제시 후 사용자 승인 없이 적용

**명령형:**
1. 사용자가 Excel 파일 업로드 → 그대로 받기 (구조 검증만)
2. FMS 연동/데이터 처리 전에 사용자에게 "구조 유지합니다"라고 확인
3. 구조 변경이 필요하면 먼저 사용자 승인 받기 (자동 처리 금지)
